此文档定义了赵涌在线App v4.8版本主要更新内容如下：


  • 注：本版需在新加价幅度上线后强制更新 —— 即：在新加价幅度上线时，低于该版本的APP打开后提示升级新版本，且无法使用

    

    # 委托出售

  • 新增 委托出售-申请单管理

   

    # 调整加价幅度

  • 提高出价体验：用户在出价输入框内输入大于下一口价且不符合加价规则的出价时，直接点击“出价”，系统将按照用户出价的符合出价规则的上一口价格出价
  • 调整加价幅度：
    • PC（上海、香港、英文）
    • APP
    • H5
  • 更改加价幅度介绍H5：http://h5.zhaoonline.com/onlineapp/index.html#/help/bidRule，需补充出价向下出符合加价规则价格的告示
  • 去除原来一键委托时针对起拍价的限制

 

   # 客户端优化及bug

  • 客户端图片限制调整：打开原图的时候增加 !D 水印
    • 预展+竞买： !D
    • 成交+其他：!2000!D
    • 优化图片渲染速度
  • 其他客户端优化：
    • 首页推荐流的藏品一次加载20条


    #服务端优化

  • 服务端优化（优化搜索体验、提高服务稳定性、减少突发宕机可能性）:
    • 解决藏品截标时：分类里一直刷新会消失的bug
    • search api 重新加密调用新的接口
    • 索引迁至阿里云
    • ES迁至阿里云
    • 商城、竞买商品合并迁移
    • search index 与app service 独立化
